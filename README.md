README
==============================

Once only setup
-------
```
npm install
```

Commands
-------
Run the WebSocket server:
```
npm run server
```

Editor
-------
Any text editor can be used, but Visual Studio Code is preferred.

Press F5 to debug the server.

Web Sample
-------

Open `index.html` in the web browser. `ws://localhost:8080` will be used as a
server.
